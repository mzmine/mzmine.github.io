User Support

The manual and various tutorials on how to use MZmine can be found here.
Help on installing MZmine can be found here.
The current list of issues and feature requests can be seen on GitHub.

If you still have questions or need help then please send and email to the developers' mailing list and we will do our best to help you out as fast as possible.

Suggestions and feature requests

Your comments are very important to us as we wish to improve MZmine all the time. We therefore encourage all MZmine users to give us input and suggestions on how we can improve the software to help solve your needs.

Add your suggestions to the request list here or send them to the developers' mailing list.

How to report issues

If you encounter a problem with MZmine then please send the below information in an email to the developers' mailing list.

  1. Your MZmine version
  2. Your OS version - is it 32-bit or 64-bit?
  3. Your Java version - is it 32-bit or 64-bit?
  4. A copy of your startMZmine script - please rename it to .txt before attaching it to the email
  5. Screenshots of all the error messages or unexpected output
  6. The log messages reported in the MZmine console window
  7. The exact steps you took until you encountered the problem. Please list all the relevant parameter values you set for each data processing step. If you feel that the results provided by MZmine were wrong, please describe exactly why you think they were wrong and how they differ from the results you expected. It will help us a lot in tracking down the problem if you upload your MZmine project file to DropSend or Dropbox so that we can use it to reproduce your steps.

  8. The following extra information if your problem is related to any MZmine modules using R: (XCMS module, baseline correction etc.)
    • Your R version - is it 32-bit or 64-bit?
    • The output from the following R commands from the R console:
      .libPaths()
      packageDescription("xcms")
      packageDescription("ptw")
      packageDescription("rJava")
